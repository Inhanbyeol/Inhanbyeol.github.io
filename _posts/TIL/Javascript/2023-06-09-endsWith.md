> endsWith()는 문자열이 특정 문자열로 끝나는지 확인한다. IE는 Edge부터 지원한다.
{: .prompt-info }

```js
const str = "hello world!";

console.log(str.endsWith('world!')) // true
console.log(str.endsWith('llo')) // false
console.log(str.endsWith('wor')) // false

```
